# Zomato Data Analysis

## Project Overview
This project analyzes Zomato restaurant data to understand restaurant trends, customer preferences, and pricing patterns. The goal is to explore the dataset and generate insights about restaurant ratings, cuisines, cost, and location distribution.

## Problem Statement
Food delivery platforms like Zomato host thousands of restaurants with different cuisines, pricing levels, and ratings. Understanding restaurant data can help identify popular cuisines, pricing trends, and highly rated restaurants.

This project analyzes Zomato data to discover patterns in restaurant ratings, cuisines, and customer preferences.

## Dataset
The dataset contains information about restaurants listed on Zomato, including:

- Restaurant Name
- Location / City
- Cuisines
- Average Cost for Two
- Rating
- Votes
- Online Delivery Availability
- Table Booking Availability

This dataset helps analyze restaurant popularity, pricing patterns, and customer engagement.

## Tools and Technologies Used
- Python – Data analysis and preprocessing
- Pandas – Data cleaning and manipulation
- Matplotlib / Seaborn – Data visualization
- Jupyter Notebook – Analysis environment

## Analysis Performed
The following analysis was performed in this project:

- Distribution of restaurant ratings
- Most popular cuisines
- Average cost for two analysis
- Restaurants with the highest votes
- City-wise restaurant distribution
- Online delivery availability analysis

## Visualizations
Several visualizations were created to better understand the dataset:

- Rating distribution chart
- Cuisine popularity chart
- Average cost distribution
- Top restaurants by votes
- City-wise restaurant count

## Key Insights
- Certain cuisines are significantly more popular than others.
- Restaurants with higher ratings tend to receive more customer votes.
- Pricing varies significantly depending on the restaurant location.
- Online delivery availability influences restaurant popularity.

## Conclusion
Zomato data analysis helps understand restaurant trends, customer preferences, and pricing patterns. These insights can help businesses make better decisions regarding pricing, location strategy, and customer service.
